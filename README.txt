================================================
Voice Dispatch - Speech Recognition
================================================

HOW TO USE:
1. Double-click "run.bat" to start the program
2. Keep this window open while playing FiveM
3. Press N key in-game to use voice dispatch

FIRST TIME SETUP:
- Edit "config.json" to match your language
- For Indian English: Change "Language" to "en-IN"
- For US English: Use "en-US" (default)
- For UK English: Use "en-GB"

Your Windows Speech Recognition language must match!
Check: Windows Settings > Time & Language > Speech

TROUBLESHOOTING:
- If accuracy is low: Train Windows Speech Recognition
- If "recognizer not found": Install language pack in Windows
- Make sure port 8765 is not blocked by firewall

For full documentation, see VOICE_COMMANDS.md
